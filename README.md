# automation-practice

Projeto de automação num site de estudos para portfólio pessoal.

### Clone o projeto

```bash
git clone https://github.com/driuzzo/automation-practice.git
```

### Instale as dependências
```bash
npm install
```

### Execute o cypress
```bash 
npx cypress open
```

### Execute o script de teste
```bash
automation-practice.spec.js
```
